# UK Car Charging Map - Angular 6

![alt text](/screenshot.png)

## Installation
```
npm install
```
## To Run Locally
This will run on localhost:4200
```
npm start
```

## To Build Production Version
This requires docker && docker-compose. Will run on localhost:80.
```
docker-compose up -d
```

## Run Unit Tests
```
npm test
```

## Run e2e Tests
This will open up cypress
```
npm run e2e
```
